> tree

```
├─assets (资源文件)
│  └─images
├─components  (组件)
│  ├─public
│  └─skeleton (骨架屏)
├─config  (配置)
├─service (服务)
├─permission.js (权限)
├─registerServiceWorker.js (PWA注册机)
├─skeleton.js (骨架屏注册)
├─index.styl (全局样式)
├─store (数据)
│  └─modules
├─utils (工具)
└─views (视图)
    ├─account (个人中心)
    ├─artboard  (服务申请)
    ├─root  (首页)
    └─sign  (注册/登录)
```
